# 3-Pass Recursive Prompting

Run a 3-pass recursive self-critique on any claim, diagnosis, plan, or analysis.

## What it does

Stress-tests thinking through three passes:

1. **Pass 1: Critique 🔍** — Tear it apart. Find logic errors, missing evidence, confirmation bias, alternative explanations.

2. **Pass 2: Refine 🔧** — Address every weakness. Fill gaps, add evidence, fix reasoning.

3. **Pass 3: Final Answer ✅** — Synthesize into a clear, defended conclusion.

## Usage

```
/skill 3pass <text to analyze>
```

Or reply to a message with `/skill 3pass` to analyze it.

## Example

```
/skill 3pass "The user model update system should store facts, not predictions"
```

Output: Critique → Refined version → Final recommendation with confidence level.

## Why use this

- Catches blind spots before they become problems
- Forces evidence-based thinking
- Reduces overconfidence in initial analyses
- Great for decisions, diagnoses, and plans
